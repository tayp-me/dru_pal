# Educator Module

This directory contains the preloaded `educator` Dru module artifacts.

- `bootstrap_module.json`: Dru bootstrap mapping (commands only; no seeded module data).
- `educator_instructions.txt`: module-local runtime behavior instructions.
- `educator_full_conversation_spec.md`: full human-readable educator module spec.
- `educator_full_conversation_spec.yaml`: machine-readable educator module spec.
- `educator_instructions.md`: module behavior contract for queue/teach/remind/remove flows.
- `dru_module_educator_export.md`: canonical SQL export source used for integration.
