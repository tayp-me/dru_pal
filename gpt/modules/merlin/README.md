# Merlin Module

This directory contains the original Merlin GPT files plus a Dru-compatible module bootstrap file.

- `merlin_instructions.txt`: direct instruction text.
- `merlin_full_conversation_spec.md`: complete technical spec.
- `merlin_full_conversation_spec.yaml`: machine-readable spec.
- `bootstrap_module.json`: Dru registry bootstrap mapping for this module.
